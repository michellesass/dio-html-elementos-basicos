O que é HTML?

(Voltar)

HTML, em inglês, significa "HyperText Markup Language" (Linguagem de marcação de hipertexto).

Criado no início da década de 1990 por Tim Berners-Lee (um físico da CERN) o HTML não é uma linguagem de programação, mas sim uma linguagem de marcação que ordena os elementos de uma página, separando-os conforme sua relevância (parágrafo, título, subtítulo etc).

Dentre suas funções, o HTML também possibilita a inserção de cores, imagens, links e listas.

Para a inserção de cores e fontes estilizadas utiliza-se a tag < font >. Ou melhor dizendo, antigamente para a inserção de cores e fontes estilizadas se utilizava a tag < font >. Atualmente, no HTML5, essa tag não é mais suportada e foi substituída por propriedades do CSS.

A essa mistura de textos com recursos de estilo e mídia dá-se o nome de hipertexto.

Como resultado é possível criar uma estrutura, um esqueleto de uma página, um conteúdo.

Estrutura básica de um HTML

(Voltar)

Todos os documentos HTML começam com uma declaração de tipo de documento: "< !DOCTYPE html >". Essa declaração geralmente aparece no início do documento HTML e ajuda os navegadores a renderizarem corretamente a página.

O documento HTML mesmo começa com < html > e termina com < /html >. Tudo o que estiver entre esse comando é um documento HTML.

A parte visível do documento HTML fica entre < body > e < /body >. Na tradução literal para o português, "body" significa "corpo". Então podemos dizer que o corpo do documento é a parte que o usuário pode ver e interagir.

Tags mais utilizadas

(Voltar)

As tags em HTML são elementos fundamentais usadas para estruturar e formatar conteúdo em uma página da web. Elas são cercadas por colchetes angulares (< e >) e geralmente aparecem em pares: uma tag de abertura e uma tag de fechamento. Por exemplo:

< p > Isto é um parágrafo < /p >

Neste caso, < p > é a tag de abertura que indica o início do parágrafo, e < /p > é a tag de fechamento que indica o final do parágrafo.

Em alguns casos as tags não possuem pares. Isto é o que chamamos de tag de autocompletar, o que significa que ela não possui uma tag de fechamento correspondente, como é o caso da < hr > usada para criar uma linha horizontal em uma página da web. Geralmente é usada para separar seções ou elementos visuais distintos. Só é preciso digitar < hr > para ela funcionar, não existindo a necessidade de um fechamento < /hr >.

Também existem as tags que utilizam atributos para o seu melhor desempenho, como o apresentado abaixo na tag < a >, utilizada para a inserção de links:

< a href = "https://web.dio.me"> Nome do link < /a >

Nesta caso, o href é responsável por especificar o destino do link. O href é o que chamamos de atributo, utilizado para fornecer informações adicionais sobre os elementos HTML.

As tags em HTML são usadas para criar diferentes tipos de elementos, como parágrafos, títulos, listas, links, formulários e muito mais, permitindo assim a criação de páginas web com estrutura e funcionalidades variadas.

Além das citadas anteriormente podemos encontrar com mais frequência as seguintes tags:

  • < h1 > até < h6 >: definem os títulos e seu grau de importância, sendo < h1 > o título mais importante e o < h6 > o menos importante.

    • Texto com a tag < h1 >

    Texto com a tag < h2 >

    Texto com a tag < h3 >

    Texto com a tag < h4 >

    Texto com a tag < h5 >
    Texto com a tag < h6 >
  • < p >: esta tag define um páragrafo. Um parágrafo sempre começa em uma nova linha e os navegadores adicionam automaticamente algum espaço em branco (uma margem) antes e depois de um parágrafo.
  • < mark >: elemento que define o texto que deve ser marcado ou destacado.
  • < small >: esta tag faz com que o texto fique escrito num tamanho menor do que o texto normal.
  • < i >: quando utilizada, essa tag deixa o conteúdo interno em itálico. A tag < i > costuma ser utilizada para indicar um termo técnico, uma frase de outro idioma ou um pensamento, por exemplo.
  • < u >: é usada para aplicar sublinhado ao texto contido dentro dela.
  • < strong >: utilizado para destacar um texto com forte importância. O conteúdo interno normalmente é exibido em negrito.
  • < ol >: "ordered list" (lista ordenada) é usada para criar uma lista na qual a ordem dos itens é importante. Os itens da lista são numerados automaticamente ou marcados com outros tipos de ordem, como letras, números ou algarismos romanos.
  • < ul >: cria uma lista na qual a ordem dos itens não possui importância significativa. Os itens da lista são geralmente marcados com pontos ou símbolos de marcadores.
  • < li >: é usada para definir um item da lista em uma lista ordenada (< ol >) ou uma lista não ordenada (< ul >). Ela representa cada item individual dentro de uma lista. Então para cada item que desejamos adicionar em uma lista é necessário abrir e fechar uma tag lista: < li > Item da lista < /li >.
  • < sub >: define o texto subscrito. O texto subscrito aparece um pouco abaixo do texto normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O
  • < sup >: define texto sobrescrito. O texto sobrescrito aparece um pouco acima do texto normal e às vezes é renderizado em uma fonte menor. Texto sobrescrito pode ser usado para indicar uma observação que será apresentada mais no final do página, como Artigo de Referência[1]
  • < blockquote > usada para indicar que o texto contido dentro dela é uma citação de outro autor ou fonte.
    "Normalmente os navegadores web irão renderizar o conteúdo dentro de < blockquote > com uma formatação específica, como um recuo em relação ao restante do texto, para indicar visualmente que se trata de uma citação."
